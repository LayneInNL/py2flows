# py2flows

A control flow generator for Python that is is able to generate control flow graphs and flows.

## Supported language features

- [x] basic assignments
- [x] if statements
- [x] while statements
- [x] for statements
- [x] IfExp(partially)
- [x] list comprehensions(partially)
- [] dict comprehensions
- [x] generator(partially)
- [] lambda
- [x] try(partially)
- [x] break
- [x] continue
- [x] pass
- [x] return
- [x] expr
- [x] call(partially)
- [x] Import
- [x] ImportFrom
- [x] Module
- [x] Remove comments and docstrings
